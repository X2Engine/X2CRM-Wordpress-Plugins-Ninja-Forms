Ninja Forms X2CRM API integration plugin

== Description ==

Ninja Forms X2CRM API integration plugin. This plugin provides an X2CRM API Action in Ninja Forms,
allowing you to send your new leads acquired with Ninja Forms into X2CRM.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Edit the form you'd like to post into X2CRM, and add a new 'X2CRM API Action' to the list of actions.
1. Configure the API action with the URL, username, and API key for your X2CRM installation, then update the field map to match your Ninja Forms fields with those in X2CRM.
